# YouTube Analytics Dashboard (Power BI)

## Project Overview

This project presents an interactive **YouTube Analytics Dashboard** built using **Power BI**. The dashboard analyzes YouTube channel performance by visualizing key metrics such as views, likes, dislikes, comments, and category-based engagement. It helps identify top-performing channels, viewer engagement patterns, and category distribution.

## Objective

The objective of this project is to transform raw YouTube data into meaningful insights using data visualization techniques. This dashboard enables users to quickly understand channel performance and audience engagement.

## Dashboard Features

* Total Views, Likes, Dislikes, and Comments KPIs
* Top Channels by Views
* Top Channels by Comments
* Views by Category
* Views by Time Frame
* Engagement comparison using Likes and Dislikes
* Interactive filters for Category ID, Time, and Channel Title

## Tools & Technologies Used

* **Power BI Desktop**
* Data Visualization
* Data Analysis
* Interactive Dashboard Design

## Key Insights

* Some channels generate significantly higher engagement and view counts.
* Viewer activity varies across different time frames.
* Certain categories contribute the majority of total views.
